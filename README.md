# AI Content Automation System

## Overview
This project is an automated AI pipeline that:

- Generates content using Ollama
- Creates images using Node.js (server.js)
- Uploads images to Cloudinary
- Posts automatically to Instagram
- Sends same content to Telegram
- Runs daily using n8n Schedule Trigger

## Tech Stack
- Node.js
- n8n
- Ollama
- Docker
- Cloudinary
- Instagram Graph API
- Telegram Bot API

## Run Locally

cd image-generator
npm install
node server.js
